# ממצאים, מסקנות ותובנות — בלשי הצלילים

**עודכן:** 2026-07-05. כל אלה נבדקו בפועל במהלך הבנייה.

## 🎤 קול והקלטה (הכי חשוב)
1. **Artifact חוסם מיקרופון.** ה-iframe המוטמע ב-claude.ai לא מאשר `getUserMedia` (`policyMic=false`, `NotAllowedError`, בלי prompt). אומת בבדיקה. → **להקלטה חייבים אתר עצמאי.**
2. **getUserMedia דורש "סביבה מאובטחת":** `https` או `http://localhost`. **`file://` (דאבל-קליק) חוסם מיקרופון.** לכן ההרצה המקומית היא דרך `python3 -m http.server`, לא פתיחת-קובץ.
3. **`localStorage` ו-`IndexedDB` כן עובדים** בתוך Artifact — רק יכולות-מכשיר (מיקרופון/מצלמה) חסומות.

## 🔤 טקסט-לדיבור (TTS) בעברית
4. **במק מותקן רק Carmit** (אין קול עברי משופר/Siri). Web Speech מנגן אותו — מכני, וגרוע יותר: **טועה בניקוד** (מנחש תנועות כי הטקסט בלי ניקוד).
5. **תיקון זול ויעיל:** להזין ל-Carmit טקסט **מנוקד** → מבטל את רוב טעויות-ההגייה. (מיושם: מפת `NIKUD` במשחק.)
6. **ElevenLabs חינם = מבוי סתום לעברית:** הקולות החינמיים אנגלית-במקור → מבטא זר. הקולות העבריים הטובים הם "library voices" ש**חסומים בתשלום** (HTTP `402 payment_required`, "Free users cannot use library voices"). ההקלטה עלתה כמה קרדיטים לבדיקה — לא שווה.
7. **מסלולים טובים לעברית טבעית (לעתיד):** Azure `he-IL-HilaNeural`/`AvriNeural` (נוירוני יליד-עברית, מכסה חינמית) · Google `he-IL-Wavenet` (יליד-עברית, דרך ה-GCP הקיים) · **הקלטה-עצמית** (הכי חם ואנושי, חינם, הגייה מושלמת).
8. **החלטה נוכחית:** בלי קול נוירוני. משוב = **צלילים** (Web Audio), הקראת-מילים = Carmit מנוקד, סטודיו-הקלטות פארק לעתיד.

## 🐛 מלכודות טכניות
9. **קובץ HTML עצמאי צריך `<meta charset="utf-8">` משלו.** ה-wrapper של Artifact מזריק אותו — בלעדיו עברית = ג'יבריש (`×‘×œ×©×™`) כשמגישים ב-localhost/host. (תוקן.)
10. **`document.title` דינמי מתוך iframe של Artifact לא מתעדכן בטאב** (רק ה-`<title>` ההתחלתי). אז אי-אפשר להשתמש בכותרת-הטאב כערוץ-דיבאג למצב-פנימי. (לקריאת מצב-iframe חוצה-מקור השתמשתי בצילום-מסך של תוכן-העמוד.)
11. **צילום-מסך אוטומטי של iframe-Artifact לא אמין** (יצא לבן שוב ושוב אף שהמשחק תקין). אימות: Node boot + מלכודת-`window.onerror` + הרצה מקומית שרנדרה נכון.

## 🎓 עקרונות-עיצוב (ממחקר, מיושמים)
12. **משוב מיידי אך חם:** טעות = לא-צורם (צבע חם, לא אדום-בּוּזֶר), הזמנה לנסות שוב + רמז. עידוד ההתמדה, לא ענישה.
13. **משבחים מאמץ/חשיבה, לא רק "נכון":** תגמול-יתר חיצוני מוריד מוטיבציה פנימית. משפטי-עידוד מגוונים + כוכב-מאמץ קטן, בלי "מכונת-מזל".
14. **עידוד עצירה-לחשוב:** אחרי טעות — נעילת-רגע "חשוב רגע…" + רמז מטא-קוגניטיבי, אין תגמול על מהירות.
15. **UX לילד שמתקשה לכתוב:** אפס-מקלדת (רב-ברירה/התאמה/לחיצה), יעדי-מגע גדולים, הקראה בכל מקום, משוב ויזואלי+קולי מיידי.
16. **עיצוב-סאונד:** כמה צלילי-הצלחה מגוונים (בחירה אקראית → לא מונוטוני) + צליל-טעות עדין ומזמין + מנגינת-סיום לשלב. הכל ב-Web Audio (בלי קבצים חיצוניים).

## 🎵 חרוזים בעברית (כלל בלשני — קריטי לנכונות)
17. **חרוז = התאמת התנועה המוטעמת + העיצורים בסוף — לא רק האות האחרונה.** באג שנתפס: כַּדּוּר (תנועת **וּ**) **לא** חורז עם אוֹר/שָׁחוֹר/צִינוֹר (תנועת **וֹ**). תמיד לבדוק את **התנועה**, לא רק את האות. → תוקן ל"חלון".
18. **זוגות בטוחים (מהתכנית, מנוקדים):** חלון-בלון-סלון-ניילון (־לוֹן) · בית-זית (־ַיִת) · נמר-מסמר (־מֵר) · שיר-עיר-קיר (־יר) · מדינה-ספינה-פינה (־ינה) · מחברת-כותרת (־ֶרֶת) · שמים-מים (־ַיִם) · ירח-אורח (־רֵחַ) · פרה-מורה-שרה (־רה) · ארוחה-מנוחה (־וחה) · תמונה-שכונה (־ונה). מקורות: ראה [SOURCES.md](SOURCES.md).

## 🔊 Web Audio — סינתזת צלילים
19. **באג-מפתח:** חייבים `o.connect(g)` (אוסצילטור→ווליום) **וגם** ווליום→יציאה. שכחתי את הראשון → כל הצלילים הטונאליים **אילמים**, ורק מבוססי-רעש (BufferSource, כבר מחוברים) נשמעו. אומת ב-`OfflineAudioContext`: 0.20 שיא עם התיקון מול 0.00 בלעדיו.
20. **סינתזה נעימה:** פעמון = הרמוניות מוערמות + אי-הרמוניות קלה (×2.01,×3.02,×4.2) · שימר = delay-feedback קל · ויברטו = LFO על התדר · attack רך ~10ms (מונע חד/צורם) · שינוי-גובה ±2% כדי שחזרות לא יישמעו רובוטיות · **עולה=הצלחה, יורד=טעות** (קונבנציית-סאונד). מימוש: `bp/bell/pluck/nz` ב-`index.html`, מעבדת-האזנה ב-`tools/sound-lab.html`.
