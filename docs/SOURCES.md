# מקורות ולינקים — בלשי הצלילים

אוסף המקורות המעולים שנמצאו ושימשו לבניית התוכן. מסודר לפי נושא, עם הערה על התועלת.

## 🇮🇱 חרוזים ומודעות פונולוגית בעברית (תוכן + תכנית לימודים)
- **משרד החינוך — דף הסבר "חרוזים" (גיל הרך)** — https://meyda.education.gov.il/files/popCharedi/files/kindergarten/Explanatory-page-rhymes.pdf ⭐ *מקור רשמי; רשימות משפחות-חרוז (־ינה: קינה/פינה/דינה...).*
- **יסוד — מדריך למורה "קוראים בשבילים" (ד"ר עמליה בר-און)** — https://www.yesod.co.il/wp-content/uploads/2018/10/madal_U3952.pdf ⭐ *מדעי; רצף מודעות פונולוגית.*
- **אורנים — "קפיצות בחרוזים" (זוגות מילים מתחרזות)** — https://www.oranim.ac.il/sites/heb/SiteCollectionImages/pictures/preschoolers-special/writing/קפיצות_בחרוזים.pdf ⭐ *רשימת זוגות מוכנה לפעילות.*
- **כרמית אלון — מודעות פונולוגית בגיל הרך** — https://www.carmitalon.co.il/post/child-development-phonologic-awareness · **חרוזי מילים (עזר למורים)** — https://www.carmitalon.co.il/files/khrvzy-mylym
- **אחיה ידע — מילים מתחרזות, דף תמונות (כיתות א–ב)** — https://achiyayeda.org/material/מילים-מתחרזות-דף-תמונות/
- **דָפִּי — דף עבודה חרוזים** — https://dapi.co.il/worksheets/דף-עבודה-חרוזים-לילדים/ *(משם: חלון-בלון, בית-זית, נמר-מסמר)*
- **ילדים.org — משחק חרוזים (PDF)** — https://yeladim.org.il/wp-content/uploads/2017/07/mishak_haruzim.pdf
- **worksheets4kids — מילים מתחרזות (גיל 5)** — https://worksheets4kids.co.il/preschool/rhymes-words-age5/
- **עפיפונים — מילים מתחרזות** — https://afifonim.co.il/product/rhyming-words/
- **Wordwall — מודעות פונולוגית כיתות א/ב** — https://wordwall.net/he-il/community/מודעות-פונולוגית-כיתות-א
- **מנוע חיפוש חרוזים בעברית** — https://brachot.net/rhymes 🔧 *כלי: מכניסים מילה ומקבלים חרוזים — שימושי לבניית תרגילים חדשים.*

## 🌍 פדגוגיה — מודעות פונולוגית (מבוסס-מחקר, אנגלית)
- **IES — Foundational Skills to Support Reading K-3** — הוראה מפורשת של מודעות פונולוגית (blending/segmenting/deletion/substitution).
- **Reading Rockets — Phonological & Phonemic Awareness** — https://www.readingrockets.org/topics/phonological-and-phonemic-awareness *(רצף מיומנויות + פעילויות לכיתה ב').*
- **NC DPI — Phonological Awareness 2nd/3rd grade** — כיתה ב-ג מתמקדת ב-blending/segmenting/deletion/substitution.

## 🔊 סאונד-דיזיין (משחקים ואפליקציות-ילדים)
- **SFX Engine — Best Practices for Game UI Sounds** — https://sfxengine.com/blog/best-practices-for-game-ui-sounds ⭐ *טונים קונסוננטיים=חיובי; עולה=הצלחה, יורד=כישלון; קצר <0.3ש', attack רך.*
- **Juicy UI (מושג של Petri Purho)** — "פלט הרבה מעל לקלט" = סיפוק רגשי.
- *(מחקר) תגמול לילדים = פופ מבריק/נצנוץ/שימר עדין — בלי עיוותים או גבוהים כואבים.*

## 🎛️ Web Audio API (סינתזה טכנית)
- **MDN — Web Audio API** — https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API
