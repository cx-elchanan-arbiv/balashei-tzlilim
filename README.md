# בלשי הצלילים 🔍

משחק מודעות פונולוגית אינטראקטיבי לכיתה ב'–ג' (עברית). הכל בלחיצות — בלי כתיבה — עם הקראה קולית, משוב חם, כוכבי-מאמץ ושמירת התקדמות.

**אתר חי:** https://cx-elchanan-arbiv.github.io/balashei-tzlilim/

## מבנה
| נתיב | מה זה |
|---|---|
| `index.html` | **כל האפליקציה** (קובץ עצמאי אחד: HTML+CSS+JS, בלי תלויות). זה מה שמתפרסם. |
| `voice/` | קבצי-קול נוירוניים (כשנבחר מנוע). |

## הרצה מקומית (עם מיקרופון להקלטות)
```
cd ~/Projects/balashei-tzlilim
python3 -m http.server 8756
```
פותחים `http://localhost:8756` — על localhost המיקרופון עובד (הקלטת קול בסטודיו).
> `file://` (דאבל-קליק) **חוסם מיקרופון** — חייבים localhost או https.

## פריסה (עדכון האתר)
```
git add -A && git commit -m "..." && git push
```
GitHub Pages בונה מחדש אוטומטית (~30ש') → האתר מתעדכן.

- **ברירת-מחדל היום:** Carmit (TTS של המערכת) **מנוקד** להגייה נכונה + **סטודיו הקלטות** (קול אנושי מקומי, גובר על ה-TTS).
- **קול נוירוני:** ElevenLabs חינם נכשל לעברית (קולות טובים חסומים בתשלום). מועמדים פתוחים: הקלטה-עצמית (הכי חם), Google he-IL Wavenet, Azure הילה/אברי.
- סדר-עדיפות בהשמעה: **הקלטה אישית ← קול נוירוני ← Carmit מנוקד.**
